Cheat Sheet!

Combat Actions:

  • Every Turn you get 1 MOVE and one Action
  • Actions:
    • Attack
    • Choke
    • Equip/Drop shield
    • Get into vehicle
    • Get up (if prone)
    • Grab
    • Human shield
    • Reload
    • Run
    • Start a vehicle
    • Stabilize*
    • Throw
    • NET action(s)
    • Use an object
    • Use a Skill
    • Drive Vehicle

Formulas:

  • INITIATIVE = REF = 1d10
  • ATTACK(Melee): ATTACKER = DEX + Weapon Skill + 1d10
  • ATTACK(Range): ATTACKER = REF + Weapon Skill + 1d10
  • DEFEND: DEFENSE = DEX + Evasion + 1d10 (Highest roll wins. Attacker wins in a tie)
  • DAMAGE = Xd6 + Bonuses(if any) - DEFENDER's AC
  • See page 172
  • ATTACK(Martial Arts)ATTACKER = DEX + Form Skill + 1d10
  • DEFEND(Martial Arts)DEFENSE = DEX + Evasion + 1d10

Netrunning:

  • ATTACK: Interface + Program ATK + 1d10
  • ATTACK(Black ICE): Interface + Black ICE ATK + 1d10
  • DEFEND: Interface + 1d10 -OR- Program DEF
  • DEFEND(Black ICE): Interface + Black ICE DEF
  • See page 201
